Superstore Sales Data Analysis Dashboard

Project Overview

This project analyzes sales performance using a retail dataset.
The goal is to identify sales trends, profitability, and customer segments.

Tools Used

- Google Sheets
- Pivot Tables
- Data Visualization
- Dashboard Design

Key KPIs

- Total Sales: $229,720
- Total Profit: $28,637
- Total Quantity Sold: 37,873
- Average Discount: 15.6%

Key Insights

- Technology category generates the highest sales
- West region produces the highest profit
- Consumer segment drives the majority of sales
- High discounts reduce profit margins
- Standard shipping has the longest delivery time
